# MoveGrid 🚀

一款遵循 **KISS 原则** 的轻量级基金网格交易提醒系统。

它不仅仅是一个简单的网格工具，更引入了 **“移动中心”** 机制，旨在让网格策略在长期波动中依然有效，避免行情单边走远后策略失效。同时内置了独立的 **定投提醒** 功能。

---

## 🌟 核心特性

-   **🎯 移动网格逻辑**：当当前价格偏离网格中心超过设定阈值（如 8%）时，系统会自动平移网格中心。这确保了无论行情如何震荡爬升或下跌，你始终在“网格内”交易。
-   **📅 周期定投提醒**：支持与网格策略并行的定投机制。例如：配置“每两周周四”提醒定投，系统会自动计算间隔并精准推送通知。
-   **🛡️ 日内防双杀（冷却期）**：同一只基金在 24 小时内如果已经触发过交易信号或网格移动，则会自动进入冷却，避免在极端震荡市中被来回收割。
-   **🔔 多渠道通知**：支持 **企业微信机器人** 和 **PushPlus**（微信图文推送），实时将买卖/定投信号送达你的手机。
-   **💾 原子级数据安全**：状态保存采用临时文件+原子替换机制，即便在写入瞬间断电，也能确保 `state.json` 的数据完整不损坏。
-   **🐳 Docker 支持**：提供一键式 Docker Compose 部署方案，支持时区同步与数据持久化挂载。
-   **📝 清爽配置**：采用 YAML 格式，支持原生 `#` 中文注释，参数配置直观简单。

## 🛠️ 快速开始

### 方式一：直接运行 (Python)

1.  **克隆项目**
    ```bash
    git clone https://github.com/你的用户名/MoveGrid.git
    cd MoveGrid
    ```
2.  **安装依赖**
    ```bash
    pip install -r requirements.txt
    ```
3.  **准备配置**
    复制配置文件模板并根据注释修改：
    ```bash
    cp config.yaml.example config.yaml
    # 编辑 config.yaml 填入你的基金代码和通知 Token
    ```
4.  **运行**
    ```bash
    # 立即检查一次
    python main.py --now
    
    # 进入定时模式（默认 15:30 和 20:00 运行）
    python main.py
    ```

### 方式二：Docker 部署 (推荐)

如果你有 Docker 环境，这是最稳定的运行方式：

```bash
docker-compose up -d
```
*系统会自动挂载当前目录下的 `config.yaml` 和 `state.json`，确保数据持久化。*

## ⚙️ 配置文件说明 (`config.yaml`)

```yaml
funds:
  - fund_code: "1.515300"       # 基金代码 (支持场内ETF代码)
    grid_spacing: 0.02          # 网格间距 (2%)
    max_position: 20            # 最大持仓份额
    trade_unit: 1               # 单次交易单位
    move_trigger: 0.05          # 偏离中心 5% 时移动网格
    invest_enabled: true        # 开启定投
    invest_interval_days: 14    # 每14天定投一次
    invest_weekday: 4           # 星期四触发
```
